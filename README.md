# 2023全国大学生电子设计竞赛 H题
## 信号分离装置设计与实现

### 项目简介
本项目为2023年全国大学生电子设计竞赛本科组H题参赛作品，基于STM32主控实现双路混合周期信号的高精度分离，完成信号叠加、FFT频域分析、DDS信号重构与波形无失真输出，满足竞赛基础要求与部分发挥指标。

### 核心任务
1. 设计并实现增益为1的模拟信号加法器，完成两路信号混合
2. 对混合信号进行采集、FFT分析，分离出原始两路信号
3. 实现正弦波/三角波识别与分离，输出波形无失真、峰峰值≥1V
4. 实现信号稳定同频显示，满足示波器观测要求

### 硬件平台
- 主控：STM32F103C8T6
- 信号源：AD9833 DDS芯片
- 模拟电路：TL082运放（加法器、电压抬升、放大电路）
- 外设：OLED显示屏、示波器、信号源

### 软件实现
- 信号采集与电压抬升处理
- FFT频域分析，实现混合信号频率分解
- 正弦波/三角波波形识别算法
- DDS信号重构与输出控制
- OLED实时频谱显示

### 核心技术
- 模拟电路：加法器、电压跟随、信号放大
- 数字信号处理：FFT变换、频域分离、波形特征识别
- 嵌入式控制：STM32外设驱动、DDS信号输出
- 信号指标：无失真分离、峰峰值≥1V、稳定同频显示

### 项目成果
1. 完成增益为1的加法器设计，实现信号精准叠加
2. 实现20kHz~100kHz混合正弦波/三角波无失真分离
3. 分离输出信号满足竞赛指标，波形稳定无漂移
4. 完成完整电子设计竞赛报告，方案论证与测试齐全
